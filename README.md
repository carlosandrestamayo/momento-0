# 🧠 Plan de Mejoramiento Momento 0 - Algoritmos y Fundamentos de Programación

---

## 🎯 Descripción

Este repositorio contiene un **plan de mejoramiento académico** diseñado para fortalecer las bases del pensamiento computacional en estudiantes de programación.

El proceso se basa en una estrategia pedagógica estructurada que incluye:

- Diagnóstico inicial  
- Intervención mediante taller  
- Evaluación final  
- Retroalimentación  

---

## 📊 Metodología

El plan de mejoramiento se desarrolla en las siguientes etapas:

1. 📄 Diagnóstico inicial (Momento 0)  
2. 📘 Estudio guiado (taller)  
3. 🧪 Evaluación final   
4. 📄 Retroalimentación   

---

## 📁 Estructura del repositorio

- 📄 `diagnostico_pre.pdf` → Evaluación diagnóstica inicial  
- 📘 `taller.md` → Material de estudio   
- 🧪 `diagnostico_post.pdf` → Evaluación final    
- 📁 `recursos/` → Imágenes y materiales de apoyo  

---

## 📘 Contenido del taller

El taller está organizado en los siguientes temas:

- 💻 Lenguajes de programación  
- 🧠 Pensamiento lógico y algorítmico  
- 🔢 Tipos de datos y variables  
- ➕ Operadores  
- 🔷 Diagramas de flujo  
- 🧪 Ejercicios tipo diagnóstico  

---

## 🚀 Instrucciones de uso

1. Estudiar el archivo `taller.md`  
2. Resolver los ejercicios propuestos en `prueba diagnostica post momento 0.pdf`

---

## ⚠️ Importante

El archivo `taller.md` no incluye respuestas.
